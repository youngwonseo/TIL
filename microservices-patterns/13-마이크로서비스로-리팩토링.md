# 마이크로서비스로 리팩토링

* 모놀리식 지옥에 빠진 상황이라면 다음과 같은 비즈니스 문제를 겪고 있음
  * 느린 전달(slow delivery)
  * 버그 투성이 소프트웨어 릴리스
  * 나쁜 확장성

## 모놀리스 -> 마이크로서비스 리팩터링 전략

* 3대 전략
  1. 새 기능을 서비스로 구현
  2. 표현 계층과 백엔드를 분리
  3. 기능을 여러 서비스로 추출해서 모놀리스를 분해
